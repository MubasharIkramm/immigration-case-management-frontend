\f0\fs24 \cf0 # Immigration Case Management System - Frontend\
\
## Overview\
\
This repository contains the frontend implementation of an Immigration Case Management System developed as part of a Software Engineering team project at George Mason University.\
\
The application provides interfaces for data entry, case review, approval workflows, and immigration case management.\
\
## Technologies Used\
\
- HTML5\
- CSS3\
- JavaScript\
- Responsive Web Design\
\
## Features\
\
- Immigration case entry forms\
- Case review interface\
- Approval workflow interface\
- User-friendly navigation\
- Responsive layouts\
- Form validation and interaction\
\
